# Zed 文档中文翻译

[Zed](https://zed.dev) 是一个内置协作和 AI 工具的开源代码编辑器，由 Zed Industries 开发。它以快速、高效著称，致力于提供极致的编辑体验。

本仓库是 Zed 官方文档的中文翻译项目，旨在帮助中文开发者更好地了解和使用 Zed。

## Zed 简介

Zed 是一个现代化的代码编辑器，主要特点包括：

- **高性能**：启动迅速，响应流畅
- **协作功能**：内置实时协作工具
- **AI 集成**：内置 AI 助手，支持多种 AI 提供商
- **开源**：完全开源，社区驱动
- **跨平台**：支持 macOS、Linux 和 Windows

## 快速开始

### 基本命令

| 操作 | macOS | Linux/Windows |
| ---- | ----- | ------------- |
| 命令面板 | `Cmd+Shift+P` | `Ctrl+Shift+P` |
| 转到文件 | `Cmd+P` | `Ctrl+P` |
| 转到符号 | `Cmd+Shift+O` | `Ctrl+Shift+O` |
| 在项目中查找 | `Cmd+Shift+F` | `Ctrl+Shift+F` |
| 切换终端 | `` Ctrl+` `` | `` Ctrl+` `` |
| 打开设置 | `Cmd+,` | `Ctrl+,` |
