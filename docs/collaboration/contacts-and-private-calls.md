# 联系人与私人通话 {#contacts}

私人通话提供在频道之外的临时协作会话。将联系人添加到您的列表中，并与一个或多个人开始通话。

## 添加联系人

1. 在“协作”面板中，点击“联系人”旁边的 `+` 按钮
1. 使用 GitHub 用户名搜索联系人。\
   _注意：您的联系人必须是现有的 Zed 用户，并且已完成 GitHub 身份验证登录流程。_
1. 您的联系人将收到通知。
   一旦他们接受，您双方将出现在彼此的联系人列表中。

## 私人通话

要启动私人通话：

1. 在“协作”面板中，点击在线联系人姓名旁边的 `...` 菜单。
1. 点击 `呼叫 <用户名>`

一旦您开始私人通话，可以通过点击“协作”面板中其他在线联系人的姓名来添加他们。

---

_私人通话的工作方式类似于[频道](./channels.md)，但不包含频道特有的功能，例如频道备注。_
