# 命令面板

命令面板是访问 Zed 中操作的主要方式。其快捷键是首先需要学习的快捷键之一：{#kb command_palette::Toggle}。

![已打开的命令面板](https://zed.dev/img/features/command-palette.jpg)

要试用它，请打开命令面板并输入 `new file`。命令列表应缩小到 `workspace: new file`。按 Return 键创建一个新缓冲区。

任何时候，当您看到包含 `zed: ...` 或 `editor: ...` 等形式的命令的说明时，这意味着您需要在命令面板中执行它们。
