# 编辑代码

Zed 提供了帮助您高效编写和修改代码的工具。本部分介绍与语言服务器协同工作的核心编辑功能。

## 本节内容

- **[代码补全](./completions.md)** — 来自语言服务器的自动补全和 AI 驱动的编辑预测
- **[代码片段](./snippets.md)** — 使用制表符插入可重用的代码模板
- **[格式化与代码检查](./configuring-languages.md#formatting-and-linting)** — 配置自动代码格式化和代码检查器集成
- **[诊断与快速修复](./diagnostics.md)** — 查看错误、警告，并应用来自语言服务器的修复
- **[多缓冲区](./multibuffers.md)** — 使用多个光标同时编辑多个文件

## 这些功能如何协同工作

当您编辑代码时，Zed 会综合来自多个来源的输入：

1. **语言服务器** 根据项目的类型和结构提供补全、诊断和快速修复
2. **编辑预测** 在您输入时建议多字符或多行更改
3. **多缓冲区** 让您可以在一次操作中跨文件应用更改

例如，您可能会：

- 使用语言服务器的重命名重构功能重命名一个函数
- 在多缓冲区中查看所有受影响文件的结果
- 使用多个光标在所有位置进行额外编辑
- 如果出现问题，立即获得诊断反馈
