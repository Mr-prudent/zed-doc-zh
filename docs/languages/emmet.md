# Emmet

Emmet 支持可通过 [Emmet 扩展](https://github.com/zed-extensions/emmet) 获得。

[Emmet](https://emmet.io/) 是一个 Web 开发者工具包，可以显著改善您的 HTML 和 CSS 工作流程。

- 语言服务器：[olrtg/emmet-language-server](https://github.com/olrtg/emmet-language-server)
