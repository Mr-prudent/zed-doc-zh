# Gleam

Gleam 支持可通过 [Gleam 扩展](https://github.com/gleam-lang/zed-gleam) 提供。要了解 Gleam，请参阅 [文档](https://gleam.run/documentation/) 或查看 [`stdlib` 参考](https://hexdocs.pm/gleam_stdlib/)。Gleam 语言服务器具有多种功能，包括跳转到定义、自动导入和[更多功能](https://gleam.run/language-server/)。

- Tree-sitter: [gleam-lang/tree-sitter-gleam](https://github.com/gleam-lang/tree-sitter-gleam)
- Language Server: [gleam lsp](https://github.com/gleam-lang/gleam/tree/main/compiler-core/src/language_server)

另请参阅：

- [Elixir](./elixir.md)
- [Erlang](./erlang.md)
