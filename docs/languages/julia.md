# Julia

Zed 中的 Julia 语言支持由社区维护的 [Julia 扩展](https://github.com/JuliaEditorSupport/zed-julia) 提供。
问题反馈请提交至：[https://github.com/JuliaEditorSupport/zed-julia/issues](https://github.com/JuliaEditorSupport/zed-julia/issues)

- Tree-sitter: [tree-sitter/tree-sitter-julia](https://github.com/tree-sitter/tree-sitter-julia)
- 语言服务器: [julia-vscode/LanguageServer.jl](https://github.com/julia-vscode/LanguageServer.jl)
