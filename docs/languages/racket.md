# Racket

Racket 支持可通过 [Racket 扩展](https://github.com/zed-extensions/racket) 使用。

- Tree-sitter: [zed-industries/tree-sitter-racket](https://github.com/zed-industries/tree-sitter-racket)

[racket-language-server](https://docs.racket-lang.org/racket-language-server/index.html) 尚未在 Zed 中支持，更多信息请参见 [Issue #15789](https://github.com/zed-industries/zed/issues/15789)。
