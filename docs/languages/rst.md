# ReStructuredText (rst)

Zed 中的 ReStructuredText 语言支持由社区维护的 [reST 扩展](https://github.com/elmarco/zed-rst) 提供。
问题反馈请发送至：[https://github.com/elmarco/zed-rst/issues](https://github.com/elmarco/zed-rst/issues)

- Tree-sitter: [stsewd/tree-sitter-rst.git](https://github.com/stsewd/tree-sitter-rst.git)
- Language Server: [swyddfa/esbonio](https://github.com/swyddfa/esbonio)
