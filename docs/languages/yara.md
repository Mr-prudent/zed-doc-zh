# Yara

Zed 中的 `Yara` 语言支持由 [Yara 语言扩展](https://github.com/egibs/yara.zed) 提供。请将问题报告至 [https://github.com/egibs/yara.zed/issues](https://github.com/egibs/yara.zed/issues)。

- Tree-sitter: [egibs/tree-sitter-yara](https://github.com/egibs/tree-sitter-yara)
- Language Server: [avast/yls](https://github.com/avast/yls)
