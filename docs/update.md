# 更新

Zed 被设计为可以自动保持更新。您可以在设置中随时更改此行为。

## 自动更新

默认情况下，Zed 会在您下次重启应用程序时检查并自动安装更新。无需任何额外操作，您将始终运行最新版本。

如果有更新可用，Zed 会在后台下载它并在重启时应用。

## 如何检查您的当前版本

要检查您正在使用的 Zed 版本：

打开命令面板 (macOS 上为 Cmd+Shift+P，Linux/Windows 上为 Ctrl+Shift+P)。

输入并选择 `zed: about`。将会弹出一个包含您版本信息的模态框。

## 如何控制更新行为

如果您想关闭自动更新，请打开设置编辑器 (macOS 上为 Cmd +,) 并在常规设置下找到 `Auto Update`。
